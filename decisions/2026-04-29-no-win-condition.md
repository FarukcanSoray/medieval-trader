# No win condition; death is the only loss

## Decision
The game has no win condition. Death is the only way to lose. The game is a sandbox with one terminal punctuation — a death screen that summarises the life. Players leave with a story, not a victory verdict.

## Reasoning
Follows from the "one trader, one life" shape and the careful-merchant fantasy: mastery is demonstrated by staying alive longer and accumulating a coherent run, not reaching a goal state. There is no victory screen.

## Alternatives considered
Director offered: target wealth, target structure, debt clock, sandbox with no end, permadeath. User chose "death only; no win condition."

## Confidence
High. Explicit choice; foundational to the game's emotional arc.
