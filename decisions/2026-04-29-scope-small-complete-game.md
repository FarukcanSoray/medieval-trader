# Scope frame is small complete game (3–6 months evening work)

## Decision
The scope frame is a small, complete game expecting 3–6 months of evening work. Not an ongoing live service, not a multi-year vertical slice. The finish line is one full economic loop, one map generator, one event system, and a single persistent save with one trader.

## Reasoning
Gates the feature budget and shapes decisions about what counts as "done" — not more variants, not more polish, one of each system.

Note: After the Scope Critic's review (which called the "What's in" list a 9–12 month shape described in a 3–6 month frame), the user chose to hold scope and accept timeline slippage rather than cut features. See `[[2026-04-29-no-cuts-slice-first]]`. The 3–6 month figure is therefore aspirational, not budgeted.

## Alternatives considered
Director offered a vertical slice frame (~4–8 weeks) and a larger commercial-curious frame (6+ months). User chose the middle option.

## Confidence
High. Explicit choice; reinforced in the brief.

## Source
Director intake, 2026-04-29; user answered "small complete game" to the scope-frame question.
