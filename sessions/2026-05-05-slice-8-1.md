# Patch slice-8 same-node arbitrage + follow-on simplifications

## Goal

Patch a kernel-collision breach: at fresh-world tick 0, symmetric pool curves gave buy=base and sell=2*base at the same node, enabling free profit without travel. Director escalated as hard design failure. Session ran the full pipeline for slice-8.1, then tighter loops for two removals (saturation gate, UI bars).

## Produced

- [[2026-05-05-slice-8-1-asymmetric-initial-demand-fill-by-tag]] -- tag-gated initial demand fill replaces uniform caps; `DEMAND_INITIAL_FILL_MULT_*` constants added to `world_rules.gd`; `world_gen.gd` extended to resolve `fill_mult` from node tag.
- `godot/world/world_state.gd` -- schema bump to v7; `from_dict` collapsed to v7-only acceptance; migration helpers `_migrate_v5_to_v6` and `_resolve_goods_for_migration` removed (~80 lines).
- `godot/travel/trade.gd` -- saturation sell gate removed from `try_sell` per [[2026-05-05-slice-8-1-1-saturation-sell-gate-removed]].
- `godot/ui/hud/node_panel.gd` -- ASCII pool bars (`[#####]<#####>`) and saturation tooltip removed per [[2026-05-05-slice-8-1-2-ascii-pool-bars-removed]].
- `godot/tools/measure_demand_drift.gd` -- headless drift measurement tool, no-trader policy, 200 seeds, samples ticks 0/100/500/2000 per [[2026-05-05-slice-8-1-measure-demand-drift-no-trader]].

## Decisions

- [[2026-05-05-slice-8-1-asymmetric-initial-demand-fill-by-tag]] -- initial demand fill asymmetric by node tag; breaks tick-0 symmetry.
- [[2026-05-05-slice-8-1-schema-v7-reject-v5-and-v6]] -- schema migration path strict to v7 only; no v5/v6 fallback.
- [[2026-05-05-slice-8-1-measure-demand-drift-no-trader]] -- measure demand reachability via headless tool, no active trader, rules out behavioral loops.
- [[2026-05-05-slice-8-1-1-saturation-sell-gate-removed]] -- remove saturation gate from sell path; preserves pillar ("prices are the window into pool memory").
- [[2026-05-05-slice-8-1-2-ascii-pool-bars-removed]] -- remove ASCII pool bars; redundant with `[N left]` and prices.
